This plugin will keep tracks of the amount of logins per user, plus it will show the last login date.

== Description ==

This plugin will keep tracks of:

1. the amount of logins per user
2. it will show the last login date.

Simple, but very effective ;)

== Installation ==

1. Download the plugin
2. Unzip it and copy the files to you plugin folder / install it at wp-admin->PLUGINS->ADD-PLUGIN
3. Enjoy! (it will configure itself).
